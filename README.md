# RawTube

Watch YouTube videos raw.

A YouTube web client that tries to be as simple as possible and provide distraction-free access to videos.

If you have a link to a video like `https://youtube.com/watch?v=...`, just replace `youtube.com` domain
with the RawTube host, for example `127.0.0.1:8000`.

YouTube oEmbed API requires the webpage to be accessed from a server,
so for local usage, use a web server, for example python http.server.
